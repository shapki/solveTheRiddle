# МДК.01.03 Разработка мобильных приложений
### Отгадай загадку

> С помощью созданного Android-приложения пользователь в случайном порядке последовательно получает 10 загадок из имеющихся в приложении (всего в приложении должно быть не менее 15-и загадок - опционально) и должен отгадать их. Выполненное задание представить в виде ссылки на GitHub

------------

   Текст загадки появляется на главном экране при нажатии на кнопку "Загадка". При нажатии на кнопку "Ответ" (становится доступной только после появления текста загадки) пользователь переходит на второй экран, на котором должен или отметить один из ответов из списка всех ответов на все загадки, или написать его в соответствующем поле. Если используется список ответов, то при каждом показе экрана он формируется в случайном порядке - опционально.

   После выбора (написания) ответа пользователь нажимает на втором экране кнопку "ПРОВЕРКА" (становится доступной только после указания ответа) и возвращается на главный экран, на котором появляется информация, содержащая его выбор (ответ) и результат - "Правильно" или "Не правильно" была решена загадка. На главном экране отражается информация о том, какая по счету загадка загадывается. Показанные загадки не должны появляться повторно в течение одной сессии.

   Также на главном экране имеется кнопка "Статистика". Данная кнопка  не доступна, пока пользователь не даст ответа на 10 загадок. При клике на данную кнопку пользователь переходит на третий экран, на котором отражается информация о количестве полученных загадок, количестве правильных и не правильных ответов. При нажатии на третьем экране на кнопку "Главная" пользователь должен вернуться на главный экран.

   После просмотра статистики пользователь имеет возможность или выйти из приложения, или открыть очередную сессию по отгадыванию загадок. 

   В качестве загадок можно использовать русские народные загадки или задания на проверку IQ
